# Control Plane

Minimal starter API for the RunCloud clone.

## Run
- `npm run dev -w apps/control-plane`

## Endpoints
- `GET /health`
- `GET /v1`
- `GET /v1/projects`
- `GET /v1/deployments`
- `GET /v1/deployments/latest`
- `GET /v1/logs` (API/server request logs, optional `project`, `limit`, `start`, `end`)
